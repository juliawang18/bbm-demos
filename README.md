Version 2.0 
=============

## Features
- **One-Player** 
  - Activities
    - Function Matching
    - Amplitude Matching
    - Frequency Matching
  - Sonification
  - Record Balance Board movement data (during activity)


- **Two-Player Activties**
  - Activities
    - Function Matching (Dual Screen)
    - Amplitude Matching (Dual Screen)
    - Frequency Matching (Dual Screen)
